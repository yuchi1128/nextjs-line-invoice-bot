### 機能

- LIFF のアプリ内で請求書を作成できる
- その請求書を LINE の友達に送信できる
- 自分が送った請求書の履歴を見ることができる

### 作成動機

- 家族や友達とお金の貸し借りをよくする
- お金を返してと言いづらいことを解消したい

### 使用技術

- Next.js（TypeScript）
- Supabase
- Prisma
- LIFF
- LINE Official Account Manager
- Vercel
